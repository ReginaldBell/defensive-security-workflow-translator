# Contributing to SecureWatch Engine

Thank you for your interest in contributing!

## How to Contribute

- Fork the repository and create your branch from `main`.
- Ensure code is well-documented and tested.
- Run `python -m pytest -q` and `npm run build` before submitting a PR.
- Follow the existing code style and structure.
- For major changes, open an issue first to discuss what you would like to change.

## Code of Conduct

Be respectful and constructive. Harassment or abusive behavior will not be tolerated.

## Reporting Issues

Open an issue with a clear description and steps to reproduce.

We appreciate your help in making SecureWatch Engine better!
